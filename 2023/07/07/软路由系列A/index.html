软路由系列-A

软路由是什么？

【ChatGPT】：

软路由是指利用通用计算机硬件（如个人电脑）和特定的软件来实现路由器功能的一种解决方案。传统的路由器是专门设计和制造的硬件设备，用于在计算机网络中转发数据包和管理网络流量。而软路由则是通过在通用计算机上安装特定的操作系统和软件来实现路由器的功能。

软路由的主要特点是灵活性和可定制性。通过使用通用计算机硬件，软路由可以方便地适应不同的需求和网络环境。它可以运行各种不同的路由软件，例如pfSense、OpenWrt、DD-WRT等，这些软件提供了丰富的功能和配置选项，可以满足各种复杂的网络需求。

软路由可以用于家庭网络、小型企业网络甚至是大型数据中心等场景。它可以提供诸如网络地址转换（NAT）、防火墙、虚拟专用网络（VPN）、负载均衡、流量控制等功能。此外，软路由还可以通过插件或扩展模块添加额外的功能，如广告拦截、内容过滤、入侵检测等。

总之，软路由是一种基于通用计算机硬件和特定软件实现的灵活、可定制的路由器解决方案，可以提供丰富的网络功能和配置选项。

软路由的功能：

  • 科学上网
  • 去广告
  • 流量控制

等等

软路由分类：

根据常见软路由设备可以分为：

  • 硬路由刷的软路由
  • x86电脑刷的软路由
  • arm芯片小盒子刷的软路由

根据软路由运行的系统可以分为：

  1. pfSense：pfSense是一款基于FreeBSD操作系统的开源软路由解决方案。它提供了广泛的网络功能，包括防火墙、虚拟专用网络（VPN）、负载均衡、流量控制等。pfSense具有友好的用户界面和强大的配置选项，适用于家庭网络和小型企业网络。
  2. OpenWrt：OpenWrt是一款基于Linux操作系统的开源软路由系统。它专注于嵌入式设备和无线路由器，并支持大量的硬件平台。OpenWrt具有灵活的软件包管理系统，可以轻松添加各种功能和扩展，如VPN、QoS、无线热点等。
  3. DD-WRT：DD-WRT是一款基于Linux操作系统的第三方固件，可用于支持多种无线路由器。它提供了许多高级功能，如VPN、QoS、无线信号增强、热点认证等。DD-WRT的用户界面相对简洁，适合有一定技术经验的用户。
  4. RouterOS：RouterOS是由MikroTik开发的专有软路由操作系统。它适用于MikroTik品牌的硬件设备，并提供了丰富的网络功能，如防火墙、VPN、负载均衡、无线网络管理等。RouterOS具有强大的配置选项和脚本功能，适用于中小型企业和服务提供商网络。

其中普通玩家最常用的是openwrt系统。

orangepi R1 plus lts 安装openwrt

官网下载openwrt镜像文件。官网已经集成了很多插件，不过最重要的出国插件并不在内，所以需要进一步安装。

openwrt系统使用balenaEtcher烧录进tf卡，插入开发板开机即可。

http://www.orangepi.cn/html/hardWare/computerAndMicrocontrollers/service-and-support/Orange-Pi-R1-Plus-LTS.html

开发板lan口（靠近中间的那个口）连接电脑，用浏览器输入192.168.2.1打开openwrt后台，默认无密码，会自动根据sd卡大小扩容至合适大小，所以第一次进入会比较慢。

第一次进入后，点击系统-管理权，设置密码，以后进入后台的密码就是这个了。

由于默认IP是192.168.2.1 ， 可能和上级路由器起冲突，建议通过终端ssh连接修改ip地址：

ssh root@192.168.2.1

yes

password

连接上之后输入

uci show network

查看ip。继续输入

uci set network.lan.ipaddr=’192.168.9.1’

修改ip地址，当然这个地址可以选自己喜好的。完成提交：

uci commit

检查:

cat /etc/config/network

注意输出内容这一段，看看是否成功：

config interface ‘lan’

​ option device ‘br-lan’

​ option proto ‘static’

​ option netmask ‘255.255.255.0’

​ option ip6assign ‘60’

​ option ipaddr ‘192.168.9.1’

重启网络：

ubus call network restart

输入之后终端应该是断了，要使用新的ip地址连接。如果没反应，戳一下香橙派侧边的重启按钮，重启一下机器即可。

将开发板的wan口（旁边那个）连接上级路由器的lan口，用浏览器进入后台，点击系统-软件包-更新列表。（终端中opkg update也是同样功能，耐心等待）

可以筛选器中输入theme看看，安装一个合适的主题包，也可以从这个网站找适合自己架构的软件。https://op.supes.top/packages/

系统-系统-语言和界面-主题 即可更改。

额外：

iptables和nftables是Linux系统中用于配置和管理网络防火墙规则的工具。它们的主要区别在于底层实现和语法结构。

  1. 底层实现：iptables是较早引入的工具，它使用内核中的netfilter子系统来处理网络数据包过滤和转发。nftables是iptables的后继者，也是一个新的网络过滤框架，旨在取代iptables。nftables使用新的内核框架nft，提供了更强大和灵活的功能。

  2. 语法结构：iptables使用基于命令行的规则集语法，规则通过表（table）、链（chain）和规则（rule）的组合来定义。每个表包含多个链，每个链包含多个规则。nftables引入了一种新的语法结构，更加简洁和一致。它使用表（table）、链（chain）和规则（rule）的层次结构，以及基于集合（set）的数据结构，使得配置更加灵活和易于管理。

  3. 性能和效率：nftables在处理规则时比iptables更加高效，因为它使用了更先进的数据结构和算法。它还提供了更好的扩展性和性能优化选项，可以处理更大规模的规则集。

  4. 功能和特性：nftables相对于iptables提供了一些新增的功能和特性，例如网络地址转换（NAT）规则的改进、更灵活的数据包匹配和操作、动态规则更新等。它还支持用户自定义的元数据和集合类型，使得规则的定义更加灵活和可扩展。

总的来说，nftables是iptables的升级版本，提供了更强大、更高效和更灵活的网络过滤功能。然而，iptables仍然被广泛使用，并且在许多Linux发行版中仍是默认的防火墙工具，因此在选择使用哪个工具时需要考虑特定的操作系统和需求。

安装完成后上传luci-app-openclash_0.45.129-beta_all.ipk并安装，完成后戳一下重启按钮重启整个设备。

  • clash启动前请先在插件设置页面的版本更新标签下确认内核编译版本，您也可以手动下载 Clash内核 后解压到 /etc/openclash/core/文件夹

下载后从overview里更新插件和内核即可：

服务-openclash-overviews下面可能有三个点跳动的点···的地方（也可能是X符号），点一下，出来几个按钮，比如“配置文件检查更新”，“插件和内核检查更新”，点一下，更新一下内核，如果网络不行就去下载自己安装。

进入软路由后台，config subscribe配置订阅，新增，添加订阅的梯子，保存配置，更新配置。

最后启动openclash，打开yacd控制面板或者dashboard控制面板选择节点。可以从overview中修改当前代理模式，也可以从插件设置中修改默认代理模式。重启一下开发板，即可全局出国。
